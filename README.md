# Trabalho-estrutura-de-dados
Trabalho entre Gabriel Augusto e João Calmon

Para a execução do projeto, é necessário estar instalado no dispositivo: compilador GCC; Subsistema do Windows para Linux (WSL); e algum Distro Ubuntu, no nosso caso foi 22.01. Para mudar as partidas que serão carregadas no sistema, é só copiar e colar os dados para o arquivo partidas.csv, ordenado pelas colunas ID, Time1, Time2, GolsTime1 e GolsTime2, com os valores separados por vírgula simples.

Os principais TADs utilizados foram: 
    Time, que é responsável por armazenar os dados de um único time por vez;
    Partida, que assim como Time, é responsável por armazenar os dados de uma única partida por vez;
    BDTime, responsável por armazenar todos os times, e processar informações referentes a mais de um time por vez;
    e BDPartida, responsável por armazenar todos os times, e processar informações referentes a mais de um time por vez;
